Remmina - The GTK+ Remote Desktop Client  1.2.0
Remmina is a remote desktop client written in GTK+, aiming to be useful for system administrators and travellers, who need to work with lots of remote computers in front of either large monitors or tiny netbooks. Remmina supports multiple network protocols in an integrated and consistent user interface. Currently RDP, VNC, NX, XDMCP and SSH are supported.

1 === BUILD PACKAGES ===
2 
3 You can build ubuntu packages with just 2 steps for Ubuntu series `yakkety`, `wily`, `xenial` and `trusty`.
4 
5 Before run the script `scripts/ubuntu-build.sh` you must have:
6  - a valid gpg key associated with your email address
7  - an account on launchpad.net where upload packages with the configured public gpg key
8  - set the two environment variables `DEBFULLNAME` and `DEBEMAIL` to your full name and your email used in your email address in the form "`<DEBFULLNAME>` <`<DEBEMAIL>`>"
9 
10 Use the flag `-d` to build a development package; if you want to build an official version manage the `debian/changelog` file by hand.
11 
12 After that follow what the script tell you to upload packages.
