Remmina - The GTK+ Remote Desktop Client  1.2.0
Remmina is a remote desktop client written in GTK+, aiming to be useful for system administrators and travellers, who need to work with lots of remote computers in front of either large monitors or tiny netbooks. Remmina supports multiple network protocols in an integrated and consistent user interface. Currently RDP, VNC, NX, XDMCP and SSH are supported.

6 ### For Ubuntu users we have an official PPA with Remmina 1.2 ###
7 https://launchpad.net/~remmina-ppa-team/+archive/ubuntu/remmina-next
8 
9 To install it, just copy and paste the following three lines on a terminal window
10 ```
11 sudo apt-add-repository ppa:remmina-ppa-team/remmina-next
12 sudo apt-get update
13 sudo apt-get install remmina remmina-plugin-rdp remmina-plugin-secret
14 ```
15 Don't forget to exit/kill al your running instances of remmina. The best way to do it is to **reboot** your PC. Or, if you cannot reboot, kill all running remmina with
16 ```
17 sudo killall remmina
18 ```
19 By default the RDP, SSH and SFTP plugins are installed. You can view a list of available plugins with `apt-cache search remmina-plugin`
20 
21 The remmina-next PPA can be used for Raspberry Pi 2 and 3, provided that you are running one of the supported *ubuntu derivatives on your Raspberry Pi.

23 ### For Debian 9 (Stretch) users, Remmina is available from the Backports repository ###
24 
25 Remmina is not available on the default Debian repositories.
26 
27 The [Debian Backports](https://backports.debian.org/Instructions/) repository must be enabled to install it, see [this blog post](https://www.remmina.org/wp/debian-the-boys-are-backport-in-town/) for more information.
28 
29 To install Remmina from Debian Backports, just copy and paste the following lines on a terminal window:
30 
31 ```
32 echo 'deb http://ftp.debian.org/debian stretch-backports main' | sudo tee --append /etc/apt/sources.list.d/stretch-backports.list >> /dev/null
33 sudo apt update
34 sudo apt install -t stretch-backports remmina remmina-plugin-rdp remmina-plugin-gnome libfreerdp-plugins-standard
35 ```

37 ### Raspberry PI with Raspbian Pixel 9.X
38 
39 On the Raspberry Pi we strongly advise you to use an ubuntu derivative, for example xubuntu, and install remmina using our ubuntu's PPA remmina-next as explained in this page.
40 
41 But you can also install remmina on Raspbian Pixel 9.X (stretch) using the officials Debian stretch backports by following the following instructions:
42 * Install dirmngr: `sudo apt install dirmngr`
43 * Add the following line at the end of /etc/apt/sources.list ```deb http://ftp.debian.org/debian stretch-backports main```
44 * Import the appropriate keys for the backports
45 ```
46 sudo gpg --keyserver pgpkeys.mit.edu --recv-key 7638D0442B90D010
47 sudo gpg -a --export 7638D0442B90D010 | sudo apt-key add -
48 ```
49 * Install remmina
50 ```
51 sudo apt update
52 sudo apt install gnome-keyring
53 sudo apt install -t stretch-backports remmina remmina-plugin-rdp
54 ```

56 ### For users with a distro that supports [Snap packages](https://snapcraft.io/), including Ubuntu ###
57 
58 _We are still working to deliver a stable Snap package, but we still have many issues with SNAP packaging. If you encounter any issues, try first an other mean of installation and only after report the problem through https://github.com/FreeRDP/Remmina/issues.
59 Snap currently lacks some functionalities, read this chapter to find more informations.
60 Please also note note that currently the SNAP package does not work under Ubuntu 17.10 + Wayland, see [issue #1414](https://github.com/FreeRDP/Remmina/issues/1414)._
61 
62 Ensure you've `snapd` installed in your system. Then in Ubuntu you can install the last release from the Software center, looking for `remmina`, otherwise you can install it from terminal with:
63 
64 ```sh
65 sudo snap install remmina
66 ```
67 Please note that this is not enough to make remmina fully functional, due to some limitations of the snap subsystem.
68 
69 Some features, for example password saving via a keyring, are disabled and *must* be manually enabled:
70 ```sh
71 sudo snap connect remmina:avahi-observe :avahi-observe # servers discovery
72 sudo snap connect remmina:cups-control :cups-control # printing
73 sudo snap connect remmina:mount-observe :mount-observe # mount management
74 sudo snap connect remmina:password-manager-service :password-manager-service # password manager
75 ```
76 Some features just don't work, for example accessing your personal ~/.ssh directory is not possible for a snap. You have to manually copy your ~/.ssh/* files under ~/snap/remmina/common/.ssh/
77 
78 If you want to install latest git revision of remmina, you can use it from the `edge` channel:
79 
80 ```sh
81 sudo snap install remmina --edge
82 ```
83 
84 Or update the current installed version with the selected channel:
85 
86 ```sh
87 sudo snap refresh remmina --channel=edge # use --channel=stable otherwise
88 ```
89 
90 Snap packages will be updated automatically and will include both latest `FreeRDP` git and latest `libssh 0.7` release (for better security).

122 ### Arch Linux based
123 
124 Install [remmina-git](https://aur.archlinux.org/packages/remmina-git) from [AUR](https://aur.archlinux.org/)
125 
126 There are also some external, not supported plugins provided by [Muflone](https://github.com/muflone) :
127 
128 * [remmina-plugin-exec](https://aur.archlinux.org/packages/remmina-plugin-exec/) A protocol plugin for Remmina to execute an external process.
129 * [remmina-plugin-folder](https://aur.archlinux.org/packages/remmina-plugin-folder/) A protocol plugin for Remmina to open a folder.
130 * [remmina-plugin-open](https://aur.archlinux.org/packages/remmina-plugin-open/) A protocol plugin for Remmina to open a document with its associated application.
131 * [remmina-plugin-rdesktop](https://aur.archlinux.org/packages/remmina-plugin-rdesktop/) A protocol plugin for Remmina to open a RDP connection with rdesktop.
132 * [remmina-plugin-teamviewer](https://aur.archlinux.org/packages/remmina-plugin-teamviewer/) A protocol plugin for Remmina to launch a TeamViewer connection.
133 * [remmina-plugin-ultravnc](https://aur.archlinux.org/packages/remmina-plugin-ultravnc/) A protocol plugin for Remmina to connect via VNC using UltraVNC viewer.
134 * [remmina-plugin-url](https://aur.archlinux.org/packages/remmina-plugin-url/) A protocol plugin for Remmina to open an URL in an external browser.
135 * [remmina-plugin-webkit](https://aur.archlinux.org/packages/remmina-plugin-webkit/) A protocol plugin for Remmina to launch a GTK+ Webkit browser.
