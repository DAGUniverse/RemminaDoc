Remmina - The GTK+ Remote Desktop Client  1.2.0
Remmina is a remote desktop client written in GTK+, aiming to be useful for system administrators and travellers, who need to work with lots of remote computers in front of either large monitors or tiny netbooks. Remmina supports multiple network protocols in an integrated and consistent user interface. Currently RDP, VNC, NX, XDMCP and SSH are supported.

9 Remmina is a feature rich Remote Desktop Application for GNU Linux(tm) and FreeBSD and possibly other Unixes.

11 ## Remmina features
12 
13 - Maintain a list of connection profiles, organized by groups
14 - Make quick connections by directly putting in the server address
15 - Remote desktops with higher resolutions are scrollable/scalable in both window and fullscreen mode.
16 - Viewport fullscreen mode: remote desktop automatically scrolls when the mouse moves over the screen edge.
17 - Floating toolbar in fullscreen mode, allows you to switch between modes, toggle keyboard grabbing, minimize, etc.
18 - Tabbed interface, optionally managed by groups.
19 - Tray icon, allows you to quickly access configured connection profiles
20 - Remote screen-shots

24 ## How to use Remmina
25 The following are typical use cases
26 ### Quick connect using RDP
27 Use this if you don't need or want to save your connection.
28 Only RDP, SSH, VNC and NX are supported in the quick connect.
29 ![Remmina Quick connect using RDP](http://www.remmina.org/images/Remmina_quick_rdp.png)
30 1. Choose the protocol
31 2. Insert the server name
32 3. Click on connect
33 ### Create a new Profile
34 1. Click on the "New Profile" icon
35 ![Remmina New Profile screenshot](http://www.remmina.org/images/Remmina_newprofile.png)
36 2. Enter your server connection details
37 ![New VNC Profile](http://www.remmina.org/images/Remmina_newvncprofile.png)
38 
39 - You can ask Remmina to execute commands before and after the connection
40 - If you use Avahi, you can search available hosts as highlighted
41 - You can group different connections.
