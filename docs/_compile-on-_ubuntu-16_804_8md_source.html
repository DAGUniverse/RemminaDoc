Remmina - The GTK+ Remote Desktop Client  1.2.0
Remmina is a remote desktop client written in GTK+, aiming to be useful for system administrators and travellers, who need to work with lots of remote computers in front of either large monitors or tiny netbooks. Remmina supports multiple network protocols in an integrated and consistent user interface. Currently RDP, VNC, NX, XDMCP and SSH are supported.

1 # Quick and dirty guide for compiling remmina on ubuntu 16.04
2 
3 These are instructions for people or software developers who want to contribute to the latest version of Remmina on Ubuntu 16.04.
4 
5 If you are an end user and you want to install the latest version of remmina, please use the "Remmina Team Ubuntu PPA - next branch", as explained on the [homepage of the wiki](https://github.com/FreeRDP/Remmina/wiki).
6 
7 By following these instructions, you will get Remmina and FreeRDP compiled under the /opt/remmina_devel/ subdir, so they will not mess up your system too much. This is ideal for testing remmina.
8 
9 You will also find the uninstall instructions at the bottom of this page.

15 **1.** Install all packages required to build freerdp and remmina:
16 ```
17 sudo apt-get install build-essential git-core cmake libssl-dev libx11-dev libxext-dev libxinerama-dev \
18  libxcursor-dev libxdamage-dev libxv-dev libxkbfile-dev libasound2-dev libcups2-dev libxml2 libxml2-dev \
19  libxrandr-dev libgstreamer0.10-dev libgstreamer-plugins-base0.10-dev \
20  libgstreamer1.0-dev libgstreamer-plugins-base1.0-dev libxi-dev libavutil-dev \
21  libavcodec-dev libxtst-dev libgtk-3-dev libgcrypt11-dev libssh-dev libpulse-dev \
22  libvte-2.91-dev libxkbfile-dev libtelepathy-glib-dev libjpeg-dev \
23  libgnutls-dev libgnome-keyring-dev libavahi-ui-gtk3-dev libvncserver-dev \
24  libappindicator3-dev intltool libsecret-1-dev libwebkit2gtk-4.0-dev libsystemd-dev
25 ```
26 **2.** Remove freerdp-x11 package and all packages containing the string remmina in the package name.
27 ```
28 sudo apt-get --purge remove freerdp-x11 \
29  remmina remmina-common remmina-plugin-rdp remmina-plugin-vnc remmina-plugin-gnome \
30  remmina-plugin-nx remmina-plugin-telepathy remmina-plugin-xdmcp
31 sudo apt-get --purge remove libfreerdp-dev libfreerdp-plugins-standard libfreerdp1 \
32  libfreerdp-utils1.1 libfreerdp-primitives1.1 libfreerdp-locale1.1 \
33  libfreerdp-gdi1.1 libfreerdp-crypto1.1 libfreerdp-core1.1 libfreerdp-common1.1.0 \
34  libfreerdp-codec1.1 libfreerdp-client1.1 libfreerdp-cache1.1
35 sudo apt-get --purge remove \
36  libfreerdp-rail1.1 libwinpr-asn1-0.1 libwinpr-bcrypt0.1 libwinpr-credentials0.1 libwinpr-credui0.1 \
37  libwinpr-crt0.1 libwinpr-crypto0.1 libwinpr-dev libwinpr-dsparse0.1 libwinpr-environment0.1 \
38  libwinpr-error0.1 libwinpr-file0.1 libwinpr-handle0.1 libwinpr-heap0.1 libwinpr-input0.1 \
39  libwinpr-interlocked0.1 libwinpr-io0.1 libwinpr-library0.1 libwinpr-path0.1 libwinpr-pipe0.1 \
40  libwinpr-pool0.1 libwinpr-registry0.1 libwinpr-rpc0.1 libwinpr-sspi0.1 libwinpr-sspicli0.1 \
41  libwinpr-synch0.1 libwinpr-sysinfo0.1 libwinpr-thread0.1 libwinpr-timezone0.1 libwinpr-utils0.1 \
42  libwinpr-winhttp0.1 libwinpr-winsock0.1
43 ```

45 **3.** Create a new directory for development in your home directory, and cd into it
46 ```
47 mkdir ~/remmina_devel
48 cd ~/remmina_devel
49 ```
50 **4.** Download the latest source code of FreeRDP from its master branch
51 ```
52 git clone https://github.com/FreeRDP/FreeRDP.git
53 cd FreeRDP
54 ```
55 **5.** Configure FreeRDP for compilation (don't forget to include -DWITH_PULSE=ON)
56 ```
57 cmake -DCMAKE_BUILD_TYPE=Debug -DWITH_SSE2=ON -DWITH_CUPS=on -DWITH_WAYLAND=off -DWITH_PULSE=on -DCMAKE_INSTALL_PREFIX:PATH=/opt/remmina_devel/freerdp .
58 ```
59 Please note that the above line will make FreeRDP install in /opt/remmina_devel/freerdp
60 
61 **6.** Compile FreeRDP and install
62 ```
63 make && sudo make install
64 ```
65 **7.** Make your system dynamic loader aware of the new libraries you installed. For Ubuntu x64:
66 ```
67 echo /opt/remmina_devel/freerdp/lib | sudo tee /etc/ld.so.conf.d/freerdp_devel.conf > /dev/null
68 sudo ldconfig
69 ```
70 
71 **8.** Create a symbolik link to the executable in /usr/local/bin
72 ```
73 sudo ln -s /opt/remmina_devel/freerdp/bin/xfreerdp /usr/local/bin/
74 ```
75 **9.** Test the new freerdp by connecting to a RDP host
76 ```
77 xfreerdp +clipboard /sound:rate:44100,channel:2 /v:hostname /u:username
78 ```
79 
80 **10.** Now clone remmina repository, "next" branch, to your devel dir:
81 ```
82 cd ~/remmina_devel
83 git clone https://github.com/FreeRDP/Remmina.git -b next
84 ```
85 
86 **11.** Configure Remmina for compilation
87 ```
88 cd Remmina
89 cmake -DCMAKE_BUILD_TYPE=Debug -DCMAKE_INSTALL_PREFIX:PATH=/opt/remmina_devel/remmina -DCMAKE_PREFIX_PATH=/opt/remmina_devel/freerdp --build=build .
90 ```
91 **12.** Compile remmina and install it
92 ```
93 make && sudo make install
94 ```
95 **13.** Create a symbolik link to the the executable
96 ```
97 sudo ln -s /opt/remmina_devel/remmina/bin/remmina /usr/local/bin/
98 ```
99 **14.** Run remmina
100 ```
101 remmina
102 ```
103 Please note that icons and launcher files are not installed, so don't search for remmina using Unity Dash.
104 
105 ## Uninstall everything
106 **1.** Remove the devel directory
107 ```
108 rm -rf ~/remmina_devel/
109 ```
110 **2.** Remove the binary directory
111 ```
112 sudo rm -rf /opt/remmina_devel/
113 ```
114 **3.** Cleanup symlinks and dynamic loader
115 ```
116 sudo rm /etc/ld.so.conf.d/freerdp_devel.conf /usr/local/bin/remmina /usr/local/bin/xfreerdp
117 sudo ldconfig
118 ```
